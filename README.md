# React Project with Vite

This project uses Vite as the build tool for a React application. Follow the instructions below to set up the development environment and get started.

## Prerequisites

- **Node.js**: Make sure you have Node.js installed. The required version for this project is `>= 18.20.4` and `npm >= 9.0.0`.

To check if you have Node.js installed, run the following command:
node -v

## Getting Started

1. Open terminal and navigate to the project directory (Where the package.json file is located)

2. Install the necessary depedencies w/

npm install

3. Start the developement server w/

npm run dev

## Conclusion

**Important**: Running 'npm install' is only required once on your PC. Next time you want to start the project, simply run 'npm run dev'

When you run 'npm run dev', a link is generated. **Ctrl + Click** on the link to open up the project on your browser.


## Tips

1. Vite supports hot-reloading, but if you make changes to the application and the browser isn't
reflecting those changes, try reloading the browser page manually.

2. Close the server by entering 'q' into your terminal
